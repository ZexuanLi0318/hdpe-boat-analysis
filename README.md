# HDPE船舶建造工艺流程详细说明

## 1. 原材料进场验收
- 检查HDPE板材规格、批次证明文件
- 验证材料是否为原生HDPE，黑色UV稳定级
- 测量厚度是否符合设计要求（通常≥8mm）
- 检查表面质量，无气泡、裂纹等缺陷

## 2. CNC切割与坡口加工
- 根据套料图纸进行数控切割
- 刀具选择：单/双刃O型铣刀
- 切割后去除毛刺，保证边缘光滑
- 对接缝开双面V型或U型坡口
- 坡口角度通常为30-45度，间隙≤3mm

## 3. 预弯/热成形
- 对需要弯曲的构件进行热成形
- 控制加热温度在120-150℃范围内
- 使用模具或线加热方式进行成形
- 成形后自然冷却至室温

## 4. 胎架定位与夹紧
- 在专用胎架上进行构件定位
- 使用夹具固定构件位置
- 确保对接缝间隙均匀一致
- 检查构件形状是否符合设计要求

## 5. 对接熔焊（主拼缝）
- 使用对接熔焊工艺连接主拼缝
- 加热板温度控制在180-220℃
- 加热时间根据板厚确定
- 保压冷却时间不少于规定值
- 焊缝质量达到母材强度的80%以上

## 6. 挤出焊接（加强结构）
- 对肋骨、舱壁等加强结构进行挤出焊接
- 焊接前对焊口开V/U型槽
- 使用专用HDPE焊条进行挤出焊接
- 焊缝连续、饱满，无气孔缺陷

## 7. 舾装焊接装配
- 焊接系缆桩、导缆器等舾装件基座
- 安装栏杆、扶手等安全设施
- 焊接设备安装座和管路支架
- 所有舾装件均采用焊接连接方式

## 8. 焊缝检验与尺寸验收
- 外观检查：焊缝形状、余高、表面质量
- 破坏性试验：T型剥离试件检测
- 尺寸检查：船体线型、主要尺寸偏差
- 密性试验：检查船体密封性能

## 9. 下水测试
- 进行倾斜试验测定重心位置
- 检查船舶浮态和稳性
- 测试所有设备运行状态
- 进行航行试验验证性能指标
